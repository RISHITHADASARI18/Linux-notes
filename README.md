Linux is a free and open-source Operating System (OS)
It acts as a bridge between the user and computer hardware
It manages CPU, RAM, storage, files, devices, and users
Linux is mainly controlled using the Terminal (Command Line)


Importance of Linux:
Free to use and modify.
Fast and lightweight.
Secure and less prone to viruses.
Stable and reliable.
Used by millions of developers and companies

Need for Linux:
Used in Cloud Computing (AWS, Azure, Google Cloud).
Runs most web servers and websites.
Used in DevOps, Cybersecurity, and System Administration.
Powers Android phones, supercomputers, and many embedded devices.
Learning Linux is essential for careers in Cloud Engineering and other IT fields.



 Linux & Open Source License

 Linux

 Linux is an Operating System (OS).
 It is free and open source.
 It manages the computer's hardware and software.
 It is fast, secure, and stable.
 Used in servers, cloud computing, Android, and supercomputers.

Open Source

 Source code is available to everyone.
 Anyone can view the code.
 Anyone can modify the code.
 Anyone can share the software.
 Developers work together to improve it.

Open Source License

 It is a legal permission to use open-source software.
 It tells users what they can and cannot do.
 It allows users to use, modify, and distribute the software.
 Linux uses the GNU GPL v2 (General Public License Version 2).
If you distribute a modified Linux kernel, you must also share its source code under the same license.
